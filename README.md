A simple and powerful desktop application for converting CSV files into Excel and PDF formats.



Built with Python, CustomTkinter, and PyInstaller.



\---



\## 🚀 Features



\- Convert CSV → Excel (.xlsx)

\- Convert Excel → PDF

\- Dark Mode UI (CustomTkinter)

\- Automatic column width adjustment

\- Clean and simple interface

\- Standalone EXE build (no Python required)
